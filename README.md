# Fusion

由纯 Rust 实现的轻量化 C2。

# 优势

现在的C2都乱七八糟的塞进去一大堆不一定会用得到，甚至不一定会有用的功能，

因为一个C2只需要做到 `命令执行`，`上传`，`下载`就可以连带着做到更多的事情，

完全不需要在塞入乱七八糟的功能，这就是本项目诞生的原因，极轻量化，化繁为简。
